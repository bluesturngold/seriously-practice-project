Hi, I'm just adding a README file because it's a best practice!

This is a JRPG project in Godot 4.6.2 with parts that are meant to be easy to re-use, i.e. spawn points and map transitions the scripts handle using their unique identifiers and drag-and-drop info in the inspector; quest giver, dialogue, and shop NPCs use simple arratys to store text and also have drag-and-drop functionality for shop items or quest rewards; items, abilities, player characters, enemies, and bosses also all follow templates and have values that can be defined in the inspector. 

It's mostly missing bulk (items, abilities, enemy stats, quests), graphics (tilesets, enemies, NPCs), and structure (narratively, and in terms of file organization...)

Placeholder graphics assets are 16x16 and come from Chequered Ink on Itch.io at https://ci.itch.io/all-game-assets.

Font is nesbox's TIC-80 Wide Font on Fontstruct at https://fontstruct.com/fontstructions/show/1388526/tic-80-wide-font used under Creative Commons license. Unmodified.
